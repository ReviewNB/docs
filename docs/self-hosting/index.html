Self Hosted ReviewNB¶

We offer self hosted ReviewNB for companies interested in running the application on their own servers. Self hosting works with GitHub.com & GitHub Enterprise Server. You retain complete control over all your data. Explore the sections below to get an overview. If you are inetersted in self hosting, fill the form and we’ll get in touch.
